About Messages#

This sections lists all protobuf messages defined by the services of the Jalapeño API Gateway.

'required', 'optional' & 'repeated'

All properties that are preceded by either the optional or required keyword do not contain the actual value, but rather pointers to those values. This is due to the implementation of proto2.

The reason for this design choice is explained in the section Why use proto2 instead of proto3?.
